# Weather App

A React application that allows users to search for any city and view current weather information using the OpenWeatherMap API.

## Features
- Search weather by city name
- Display current temperature, humidity, and weather conditions
- Weather icons (sun, rain, clouds)
- Error handling for invalid city names

## Tech Stack
- React
- JavaScript
- CSS
- OpenWeatherMap API
